== Required software environment ==

- Window 7 **64bit** operating system is required. 

- Install Visual Studio 2010 Professional. 

- Install **IPP 7.0** (Run w_ipp_7.0.4.196.exe)

- Extract px14400win2.1-64bit.zip and install **Signatec PX14400** digitizer 64bit driver. 

- Extract Qt.7z to C:\

- Extract boost_1_49_0.zip to Erasmus2\Library (Check Erasmus2\Library\boost_1_49_0\index.html)

- Install Matlab Compiler Runtime (MCRInstaller.exe: optional)

== Updating environment variables ==

- Set environment variable QTDIR = C:\Qt\4.8.2

- Add %QTDIR%\bin to environment variable PATH

- Add C:\Program Files (x86)\Intel\ComposerXE-2011\redist\intel64\compiler to environment variable PATH
